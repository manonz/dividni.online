    Create Lambda Expression

    Please use this feature if you have programming knowledge and/or experience with lambdas. The code should be written in C#.

    Create a lambda that takes no parameters and returns a value (either string, int, or double) that will be assigned to the variable. If necessary, you can reference other variables by using their names. See example in placeholder for more details.

    To reference a variable, start by typing the `@` sign & choosing from the list of available options. (Please note: in order for this feature to work, there must be a blank space after any other characters in front of the `@` . If necessary, you may delete any leading or trailing whitespace after the variable has been correctly selected.)
